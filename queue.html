What is a Queue?

A queue is a linear data structure following First In — First Out (FIFO). Items are added at the rear and removed from the front. Imagine a line at a ticket counter!

Enqueue: add an item to rear (O(1))
Dequeue: remove item from front (O(1))
Peek: view front without removing
Applications: print queue, CPU scheduling, BFS traversal

Pseudocode

initialize queue as array
front ← 0
rear ← -1

function ENQUEUE(x):
  rear ← rear + 1
  queue[rear] ← x

function DEQUEUE():
  if rear < front:
    error "underflow"
  value ← queue[front]
  front ← front + 1
  return value

function PEEK():
  if rear ≥ front return queue[front]
