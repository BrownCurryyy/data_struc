What is a Binary Tree?

A binary tree is a hierarchical data structure where each node has up to two children: left and right. It’s used in searching, sorting, and hierarchical data representation.

Insert: Add a node in the tree
Delete: Remove a node while maintaining structure
Traversals: Inorder, Preorder, Postorder, BFS
Applications: Expression trees, file systems, BSTs

Pseudocode

class Node:
  data, left, right

function INSERT(root, value):
  if root is null:
    return new Node(value)
  if value < root.data:
    root.left = INSERT(root.left, value)
  else:
    root.right = INSERT(root.right, value)
  return root

function SEARCH(root, value):
  if root is null or root.data == value: return root
  if value < root.data: return SEARCH(root.left, value)
  else return SEARCH(root.right, value)
