What is a Graph?

A graph is a collection of nodes (vertices) connected by edges. Graphs can be directed or undirected, weighted or unweighted.

BFS: Level-wise traversal using a queue
DFS: Depth-first traversal using recursion or stack
Applications: Social networks, maps, web crawlers

Pseudocode

function BFS(graph, start):
  queue = [start]
  visited = {start}
  while queue not empty:
    node = queue.pop(0)
    visit(node)
    for neighbor in graph[node]:
      if neighbor not in visited:
        visited.add(neighbor)
        queue.push(neighbor)
