What is a Stack?

A stack is a linear data structure that follows the Last In — First Out (LIFO) principle. Items are added and removed only from the top. Stacks are used for function call management, undo history, expression parsing, and more. Visual learners: think of a stack of trays — you put trays on top and take them from the top.

Push: add an item on top (constant time).
Pop: remove the top item (constant time).
Peek: view top without removing.
Applications: recursion, backtracking, parsing, undo stacks.

Pseudocode

/* array-based stack */
initialize stack as array
top ← -1

function PUSH(x):
  top ← top + 1
  stack[top] ← x

function POP():
  if top == -1:
    error "underflow"
  value ← stack[top]
  top ← top - 1
  return value

function PEEK():
  if top != -1 return stack[top]
